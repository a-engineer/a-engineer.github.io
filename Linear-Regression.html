Linear Regression

Linear regression equation: y = mx + b + e, where b is the y-intercept, m is the slope or the regression coefficient, x is the value of the predictor variable and e is the error term. Linear regression looks at the relationship between a response (dependent variable, y) and a explanatory variable (independent variable, x). Linear regression finds the “line of best fit” to explain this relationship.

An example of a linear regression line is: y = 10 + 5x. The value of \(b_0 = 10\) means that when the predictor variable x is zero, the predicted value will be 10 units. This is useful to know if you are interested in the initial value when x = 0. The value of \(b_1 = 5\) means that a one unit increase in x is associated with 5 unit increases in y.

Predictions can be made using the regression line. Say for instance we have a dataset that ranges in x values from 100 - 200 and we found the regression line to be y = 2x +4. Let’s assume that the x value units is speed in mph and y is the height in ft. I can make a prediction of height given the speed. Say for instance that we wanted to predict the height after 150 mph, then we would have: \(y = 2(150) + 4 = 304 ft\). However, when making predictions we must only use predictor values that are within the range of the predictor variables in the original dataset so for this example, we can only use x values from 100 - 200 mph and can not use anything small or greater than this range. .

Assumptions

There are several assumptions associated with linear regression:

  1. Linearity - the relationship between X and the mean of Y is linear.
  2. Homoscedasticity - The variance of residuals is the same for any value of X.
  3. Independence - Observations are independent of each other.
  4. Normality - For any fixed value of X, Y is normally distributed.
  5. Multicollinearity - Little to no multicollinearity is present.
  6. Sample Size - Regression analysis requires 20 cases per independent variable.

If any of these assumptions are violated then the output of the linear regression may be unreliable!

Linearity

Linear regression is sensitive to outliers so linearity must be checked by plotting the data by using a scatter plot. The data must have a linear relationship, meaning that as one variable increases or decreases the other variables must form a line with that movement. In other words, if the points on the graph appear to fall in a somewhat straight line, then the data is linear. Some data can have an exponential relationship but not a linear relationship. Therefore, one assumption of linear regression is that there is a linear relationship between the independent variable, x, and the dependent variable, y.

If there is no linear relationship then a nonlinear transformation can be applied to the independent and/or dependent variable in the form of a log, square root or reciprocal of the independent and/or dependent variable. Adding another independent variable can also assist to make the data more linear.

Independence

Independence in linear regression means that there is no correlation between residuals in the time series data, that is, one does not depend on another. To test for independence, the Durbin-Watson test can be administered.

The Durbin-Watson test uses \(H_0\) (null hypothesis): there is no correlation among the residuals and \(H_A\) (alternative hypothesis): the residuals are autocorrelated. The following performs the Durbin-Watson test for independence using the mstcars dataset.

data(mtcars) # Load the mtcars dataset
head(mtcars) # Shows the first 6 rows of data.

For this dataset, the predictor variable will be mpg, and the explanatory variables will be disp (displacement in cu. in.) and wt (weight in 1,000 lbs).

# Adds a regression model
model <- lm(mpg ~ disp+wt, data = mtcars)

# The Durbin-Watson test is found in the car library
library(car)
## Loading required package: carData
# The following applies the Durbin-Watson test on the model.
durbinWatsonTest(model)
##  lag Autocorrelation D-W Statistic p-value
##    1        0.341622      1.276569   0.016
##  Alternative hypothesis: rho != 0

From the output above we can see that the p-value is less than 0.05 so we can reject the null hypothesis and conclude that the residuals in the regression model are autocorrelated.

Homoscedasticity

This implies that the residuals have constant variance for every x value. That is, the same variance or “noise” is the same across all values of the independent variable. Simply put, homoscedasticity means that all the points must be around the same distance from the regression line but if the data points have various distances from the regression line then it is known as heteroscedasticity. A general rule of thumb is, if the ratio of the largest variance to the smallest variance is 1.5 or below, the data is homoscedastic.

Normality

Normality checks that the residuals of the model are normally distributed.

High t-values indicate that the variable should be kept in the model, variables with low t-values can be dropped from the regression model.

\(R^2\) - Coefficient of Determination

\(R^2\), known as the coefficient of determination or in this case, the multiple r-squared value, tells us how well the data “fits” the line. This value is the proportion of the variance in the response variable that can be explained by the predctor variable. A value of 0 means that the response variable cannot be explained by the predictor variable while a value of 1 means that the response variable explains the predictor variable perfectly and without error. For the data summary shown above we have a value of 0.332 which means that 33.2% of the variance in the response variable can be explained by the predictor variable.

Confidence intervals

Even though we found the regression equation, there is some error associated with that line bound by a confidence interval. To find the actual values for the slope, m, and the y-int, b, we use confidence intervals.

confint(model)
##                 2.5 %     97.5 %
## (Intercept) 9.8419062 12.5060253
## facebook    0.1622443  0.2427472

This tells us that that there is a 97.5% confidence that the slope, m or facebook value, is within [0.1622443, 0.2427472]. The true y-int value is found within the interval [9.8419062, 12.5060253] with a 97.5% confidence.

The gray bar surrounding the regression line is the confidence interval.

Model Accuracy

Once we have found the regression line it is time to determine how well the line fits the data, this is known as goodness-of-fit. The three parameters that help to determine goodness-of-fit are the residual standard error (RSE), the r-squared value, and f-statistic.

Residual standard error: 5.13 on 198 degrees of freedom Multiple R-squared: 0.332, Adjusted R-squared: 0.3287 F-statistic: 98.42 on 1 and 198 DF, p-value: < 2.2e-16

This model’s r-squared value is only 0.332 which means that the regression line only explained 33% of the variability, which is low. This makes sense since the correlation was only 57%.
